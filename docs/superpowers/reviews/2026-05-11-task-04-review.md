## Edge Cases

- **No filtered list (direct URL)**: `currentIndex = -1`, both buttons disabled ✅
- **First project**: `prevId = undefined`, `nextId = valid` ✅
- **Last project**: `nextId = undefined`, `prevId = valid` ✅
- **Single project**: both disabled ✅

## Component Integration

```typescript
<PrevNextNav
  prevId={prevId}
  nextId={nextId}
  currentIndex={currentIndex >= 0 ? currentIndex + 1 : 0}  // 1-based
  total={filteredProjectIds.length}
  onPrev={() => prevId && navigate(`/project/${prevId}`)}
  onNext={() => nextId && navigate(`/project/${nextId}`)}
/>
```
