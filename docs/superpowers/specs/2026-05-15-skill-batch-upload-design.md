# Skill 批量上传功能设计

**日期：** 2026-05-15
**状态：** 已批准

## 概述

当前 Skill 上传仅支持单个文件文本输入，用户无法批量导入多个 Skill ZIP 包。本设计新增多文件选择和拖拽上传功能，提升用户体验。

## UI 设计

### 上传区域
- 拖拽区域：点击或拖拽 ZIP 文件到此处
- 支持多文件同时选择
- 视觉反馈：悬停时边框高亮

### 文件列表
- 显示文件名和大小
- 每项有复选框可勾选
- 单独移除按钮 (✕)
- 全选/取消全选控制

### 操作按钮
- 添加更多：打开文件选择器
- 导入选中：批量导入已勾选文件

### 状态反馈
- 导入成功：绿色提示，显示数量
- 导入失败：红色提示，显示原因

## 交互流程

1. 用户点击或拖拽文件到上传区域
2. 文件列表显示已选文件，支持勾选
3. 点击"导入选中"处理已勾选文件
4. 解析每个 ZIP 包提取 prompt 内容
5. 显示成功/失败反馈

## 数据处理

- 解析 ZIP 包（使用 JSZip 库）
- 提取 `.txt` 或 `.md` 文件作为 prompt 内容
- 生成 Skill 记录存储到数据库

## 技术方案

- 前端：原生 File API + JSZip
- 无需新增后端接口
- 复用现有 Skill 数据模型

## 组件修改

- SkillPanel 组件新增拖拽上传区域
- 新增文件选择列表组件
- 添加 JSZip 依赖
