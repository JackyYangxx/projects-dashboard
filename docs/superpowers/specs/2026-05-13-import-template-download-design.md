# 导入模版下载功能设计

日期：2026-05-13

## 概述

在首页项目概览的"导入"按钮增加下拉菜单，新增"下载导入模版"选项。用户点击后可下载包含所有字段（必填+可选）的 Excel 模版文件，方便了解导入格式和必填字段要求。

## 功能位置

Dashboard 页面头部按钮区域：
- 现有"导入"按钮右侧增加下拉箭头（caret 图标）
- 点击展开菜单显示两个选项：导入项目、下载导入模版

## 模版内容

| 列名 | 说明 |
|------|------|
| 项目名称* | 必填 |
| 产品线* | 必填 |
| 负责人* | 必填 |
| 总预算* | 必填 |
| 已用预算* | 必填 |
| 代码仓 | 可选 |
| 分支 | 可选 |

- `*` 标记表示必填字段
- 第一行（表头）必填字段用 `*` 后缀标记
- 第二行（示例数据）必填字段单元格添加灰色背景色

## 技术实现

- **生成库**：使用项目已有 `xlsx` 库
- **数据来源**：复用 `src/constants/project.ts` 中的 `IMPORT_REQUIRED_HEADERS` 和 `IMPORT_OPTIONAL_HEADERS` 常量
- **文件命名**：`导入模版.xlsx`
- **下载方式**：通过 Blob + URL.createObjectURL 触发浏览器下载

## 组件变更

### Dashboard.tsx

1. 将"导入"按钮改为 SplitButton 样式（按钮 + 下拉箭头）
2. 添加下拉菜单状态管理
3. 新增 `handleDownloadTemplate` 函数生成并下载模版
4. 点击空白处关闭菜单

### 样式

- 下拉箭头：Material Symbols `expand_more`，12px
- 菜单项：hover 态背景色 `#F5F5F5`
- 菜单使用绝对定位，z-index 确保在内容之上

## 交互流程

1. 用户点击"导入"按钮 → 展开下拉菜单（100ms fade-in）
2. 选择"下载导入模版"→ 生成 Excel → 浏览器自动下载
3. 点击其他区域或再次点击"导入"按钮 → 关闭菜单
