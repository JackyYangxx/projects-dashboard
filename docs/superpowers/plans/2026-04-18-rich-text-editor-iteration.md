## 概述

## 技术方案

### 方案选型：Quill + react-quill

- **Quill** 是成熟的轻量级富文本编辑器，API 简洁，工具栏开箱即用
- **react-quill** 提供 React 封装，集成方便
- 相比 TipTap，Quill 工具栏更完整，链接/图片插入有原生弹窗

### 安装依赖

```bash
npm install react-quill
```

### Quill 工具栏配置

```javascript
modules = {
  toolbar: [
    [{ 'header': [1, 2, false] }],
    ['bold', 'italic', 'underline', 'strike', 'blockquote'],
    [{ 'list': 'ordered' }, { 'list': 'bullet' }],
    ['link', 'image'],
    ['code-block', 'clean']
  ]
}
```
