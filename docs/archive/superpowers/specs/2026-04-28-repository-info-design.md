# 代码仓信息功能设计

## 概述

为项目详情页添加代码仓和分支信息字段，支持在项目创建、编辑、导入导出时管理代码仓信息。

## 数据模型

### Project 类型变更

```typescript
interface Project {
  // ... existing fields
  repository: string  // 代码仓 URL
  branch: string      // 分支名
}
```

- `repository`: 完整 URL 或 repo 名称，可为空字符串
- `branch`: 分支名称，可为空字符串

## UI 设计

### ProjectDetail 查看态

位置：在"进度跟踪"卡片上方添加代码仓信息卡片（12列占满）

```
┌─────────────────────────────────────────────────────────────┐
│ 🗂 代码仓信息                                               │
├─────────────────────────────────────────────────────────────┤
│ repo-name @ branch-name                                     │
│ (或 "未设置代码仓" 灰色文字)                                 │
└─────────────────────────────────────────────────────────────┘
```

### ProjectDetail 编辑态

```
┌─────────────────────────────────────────────────────────────┐
│ 🗂 代码仓信息                          [保存] [取消]         │
├─────────────────────────────────────────────────────────────┤
│ 代码仓   [________________________________________________] │
│ 分支     [________________________________________________] │
└─────────────────────────────────────────────────────────────┘
```

- 两个输入框：代码仓 URL + 分支名
- Enter 保存，ESC 取消
- 保存后显示格式化文本

### ProjectForm（新增项目）

表单添加两个选填输入框：

```
┌─────────────────┐
│ 代码仓           │
│ [_____________] │
├─────────────────┤
│ 分支             │
│ [_____________] │
└─────────────────┘
```

## 导入导出

### CSV 导入

- 列名：`代码仓`、`分支`
- 非必填，缺失则为空字符串
- 位置：负责人之后

### CSV 导出

- 两列均输出到 CSV
- 空白则留空

## 实现步骤

1. 修改 `src/types/index.ts` - 添加 `repository` 和 `branch` 字段
2. 修改 `src/constants/project.ts` - 添加导入可选列 `代码仓`、`分支`
3. 修改 `src/pages/ProjectDetail.tsx` - 添加代码仓信息展示/编辑卡片
4. 修改 `src/pages/ProjectForm.tsx` - 添加代码仓和分支输入框
5. 修改导入/导出逻辑 - 处理新字段
