## Feature #1: 项目笔记仅在编辑状态下显示

**优先级：** 🟡 中等

**需求描述：**
在查看状态下，不需要显示项目笔记；只有在编辑状态下才显示。

**预期行为：**
- 查看模式：隐藏项目笔记区域
- 编辑模式：显示项目笔记编辑区域

**✅ 已实现 (2026-04-18)：**
- 项目笔记区域外层包裹 `!isReadOnly &&` 条件渲染
- 提交 `d9df988`

---

## Feature #2: 里程碑卡片支持编辑状态输入

**优先级：** 🟡 中等

**需求描述：**
里程碑卡片需要在编辑状态下支持输入，填写的里程碑在查看状态下可见。

**技术要求：**
- 使用业界成熟组件库，如 shadcn/ui
- 编辑状态：显示输入表单
- 查看状态：只读展示里程碑列表

**建议方案：**
1. 评估 shancn/ui Accordion 或类似组件
2. 编辑状态使用 Dialog 或 Sheet 模式输入
3. 查看状态使用简洁的列表展示

**✅ 已实现 (2026-04-18)：**
- 编辑模式下显示"添加里程碑"按钮
- Dialog 表单包含标题/日期/状态/描述
- 提交 `f66a113`

**✅ 已实现 (2026-04-18)：**
